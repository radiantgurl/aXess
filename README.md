# **About This Mod**

Have you ever wanted to have a giant SCP-foundation type base with thousands of employees, all of whom have different access levels to different rooms in this facility but there are no mods that offer extensive customizability, QoL and usability?? …. No? Well, maybe I was too specific.

 

***

# CUSTOMIZE EVERYTHING

Just create your network, add the access levels, give them cool names, colors and icons and then you're set!

![CUSTOMIZE EVERYTHING](https://i.imgur.com/PjLLcQb.gif)

# EASY TO SET UP

Just place two readers, set their access level, link them together, and done!

![EASY TO SET UP](https://i.imgur.com/l8HBw1w.gif)

# IT'S WIRELESS!

Stop worrying about connecting annoying redstone to the reader. Just link it to a receiver and it can activate wirelessly!

![IT'S WIRELESS!](https://i.imgur.com/n4ZSXFK.gif)

# CONFIGURE ON THE FLY

Want to make changes? Everything will respect them! Changing names, icons, priority will update everything!

![CONFIGURE ON THE FLY](https://i.imgur.com/Qa3xnUa.gif)
